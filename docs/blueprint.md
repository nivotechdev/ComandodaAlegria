# **App Name**: Joy Command Landing

## Core Features:

- Dynamic Hero Section: Eye-catching hero section with a prominent title, engaging subtitle, and a pulsing CTA button for immediate user engagement.
- Interactive Character Catalog: Interactive grid of character cards that showcases the variety of available characters. Each card includes a photo, character name, and a 'Premium' badge to highlight special offerings.
- Service Highlights: Display the 3 core service benefits (punctuality, luxury costumes, and trained staff) in a clear, concise, and visually appealing manner using a three-column layout.
- Area of Operation Messaging: Clearly communicate the service's mobile nature, emphasizing that the company brings the fun to various locations, including residences, event halls, and corporate events.
- Conversion-Optimized Footer: Include social media links and a floating WhatsApp button to ensure it's always visible on the screen.

## Style Guidelines:

- Primary color: Sky Blue (#4fb9e0), mirroring the sky and adding a playful touch.
- Secondary color: Water Green (#4db6ac) evokes freshness, vitality, and connection with nature.
- Accent colors: Vibrant Orange (#f5a623) for highlights and call to action elements and Pink (#d81b60) to signal high energy and fun.
- Font: 'Poppins' for both headings and body text. It provides a modern and geometric style which works well for headings and shorter length copy, given its avant-garde fashionable style. A single font styling gives a modern minimalist impression.
- Incorporate soft shadows, rounded-2xl or 3xl borders, and friendly icons throughout the layout to enhance the user experience with a touch of fun.
- Employ cheerful icons representing different characters and services. Make sure they fit the friendly rounded style that appears throughout the layout.
- Add subtle hover states on character cards and smooth transitions between page sections to improve interactivity.
